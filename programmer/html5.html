HTML5 标签列表

这里列出了所有标准化的 HTML5 元素，使用起始标签描述，按照功能分组。与列出所有标准化的、非标准化的、有效的、废弃的标签的 HTML 元素索引 不同的是，该页只列出有效的 HTML5 元素。新网站应当只使用这里列出的元素。

符号 代表该元素是在 HTML5 中新增的。另外注意，这里列出的其他元素可能在 HTML5 标准中得到了扩充或经过修改。

属性只显示HTML5支持并且非试验阶段的属性。并不含各属性的兼容性说明。局属性是所有HTML元素共有的属性;它们可以用于所有元素，尽管属性可能对某些元素没有影响。

根元素EDIT

    脚本EDIT
    <script> 定义一个内联脚本或链接到外部脚本。脚本语言是 JavaScript。
    属性 值 描述
    type必选的属性 MIME-type 指示脚本的 MIME 类型。
    async async 规定异步执行脚本（仅适用于外部脚本）。
    charset charset 规定在外部脚本文件中使用的字符编码。
    defer defer 规定是否对脚本执行进行延迟，直到页面加载为止。
    language script 不赞成使用。规定脚本语言。请使用 type 属性代替它。
    src URL 规定外部脚本文件的 URL。
    xml:space preserve 规定是否保留代码中的空白。
    <noscript> 定义当浏览器不支持脚本时显示的替代文字。

    全局属性。
    <template>

    通过 JavaScript 在运行时实例化内容的容器。HTML 模板元素 template 是一种机制，允许包含加载页面时不渲染、但又可以随后通过 JavaScript 实例化的客户端内容。可以将模板视作为存储在页面上稍后使用的一小段内容。尽管不会渲染，但解析器会在加载页面的时候处理 template元素中的内容来确保它是有效的。

    除此以外，还包含只读的 content 属性，通过它可以读取模板内容。一般来说，可以通过判断 content 属性是否存在来判断浏览器是否支持 template 元素。
    章节EDIT
    <body> 代表 HTML 文档的内容。在文档中只能有一个元素。

    onafterprint ，用户打印完成当前文档后的回调函数。

    onbeforeprint ,用户请求打印当前文档时（打印前）的回调函数。

    onbeforeunload ,当文档即将被卸载的回调函数。

    onblur ,DOM失去焦点的回调函数。

    onerror ,DOM加载失败的回调函数。

    onfocus ,DOM接收到焦点focus的回调函数。

    onhashchange ,当文档当前地址的片段标识部分（地址后面以＃号开始的部分）变化时的回调函数。

    onlanguagechange 试验中,语言改变时的回调函数。

    onload ,DOM完成加载的回调函数。

    onmessage ,DOM接收到message的回调函数。

    onoffline ,网络连接失败的回调函数。

    ononline ,网络恢复连接的回调函数。

    onpopstate ,用户导航历史会话的回调函数。

    onredo ,用户从历史记录里面前进的

    onresize ,DOM大小调整时触发的回调函数。

    onstorage ,存储区改变的回调函数。

    onundo ,用户从历史记录里前进时触发的回调函数。

    onunload ,当文档已经被关闭的回调函数.
    <section> 定义文档中的一个章节。
    <nav> 定义只包含导航链接的章节。
    <article> 定义可以独立于内容其余部分的完整独立内容块。
    <aside> 定义和页面内容关联度较低的内容——如果被删除，剩下的内容仍然很合理。
    <h1>,<h2>,<h3>,<h4>,<h5>,<h6> 标题元素实现了六层文档标题，<h1> 是最大的标题，<h6> 是最小的标题。标题元素简要地描述章节的主题。
    <header> 定义页面或章节的头部。它经常包含 logo、页面标题和导航性的目录。
    <footer> 定义页面或章节的尾部。它经常包含版权信息、法律信息链接和反馈建议用的地址。
    <address> 定义包含联系信息的一个章节。
    <main> 定义文档中主要或重要的内容。
    组织内容EDIT
    <p> 定义一个段落。
    <hr> 代表章节、文章或其他长内容中段落之间的分隔符。
    <pre> 代表其内容已经预先排版过，格式应当保留 。
    <blockquote> 代表引用自其他来源的内容。 cite,是一个标注引用的信息的来源文档或者相关信息的URL。通常用来描述能够解释引文的上下文或者引用的信息。
    <ol> 定义一个有序列表。
    属性 值 描述
    reversed reversed 规定列表顺序为降序。(9,8,7...)
    start number 规定有序列表的起始值。
    type
  • 1 默认值。数字有序列表。（1、2、3、4）
  • a 按字母顺序排列的有序列表，小写。（a、b、c、d）
  • A 按字母顺序排列的有序列表，大写。（A、B、C、D）
  • i 罗马字母，小写。（i, ii, iii, iv）
  • I 罗马字母，大写。（I, II, III, IV）
    		•  规定在列表中使用的标记类型。
    <ul> 定义一个无序列表。
    <li> 定义列表中的一个列表项。 value,这个整数属性显示的当前顺序值列表项所定义的 ol 元素。这个属性只允许的值是一个数字,即使与罗马数字或字母显示列表。遵循这一继续编号的列表项价值。价值属性没有意义为无序列表( ul )或菜单(菜单)。 注意:这个属性在HTML4被弃用,但在HTML5引入。 注意:壁虎9.0之前,消极的价值观不正确转换为0。从壁虎9.0整数值都是正确地解析。
    <dl> 定义一个定义列表（一系列术语和其定义）。
    <dt> 代表一个由下一个 <dd> 定义的术语。
    <dd> 代表出现在它之前术语的定义。
    <figure> 代表一个和文档有关的图例。
    <figcaption> 代表一个图例的说明。
    <div> 代表一个通用的容器，没有特殊含义。
    文字形式EDIT

    属性 值 描述

    hreflang language_code 规定被链接文档的语言。
    media media_query 规定被链接文档是为何种媒介/设备优化的。
    rel text 规定当前文档与被链接文档之间的关系。
    target
  • _blank 在新窗口中打开被链接文档。

    type MIME type 规定被链接文档的的 MIME 类型。
    <em> 代表强调 文字。
    <strong> 代表特别重要 文字。
    <small> 代表注释 ，如免责声明、版权声明等，对理解文档不重要。
    <s> 代表不准确或不相关 的内容。
    <cite> 代表作品标题 。
    <q> 代表内联的引用 。 cite,这个属性的值是URL，意在指出被引用的文本的源文档或者源信息. 这个属性重在解释这个引用的参考或者是上下文.
    <dfn> 代表一个术语包含在其最近祖先内容中的定义 。
    <abbr> 代表省略 或缩写 ，其完整内容在 title 属性中。
    <data> 关联一个内容的机器可读的等价形式 （该元素只在 WHATWG 版本的 HTML 标准中，不在 W3C 版本的 HTML5 标准中）。 value,该属性指定元素内容对应的机器可读的翻译。
    <time> 代表日期 和时间 值；机器可读的等价形式通过 datetime 属性指定。
    属性 值 描述
    datetime datetime 规定日期 / 时间。否则，由元素的内容给定日期 / 时间。
    pubdate pubdate 指示 <time> 元素中的日期 / 时间是文档（或 <article> 元素）的发布日期。
    <code> 代表计算机代码 。
    <var> 代表代码中的变量 。
    <samp> 代表程序或电脑的输出 。
    <kbd> 代表用户输入 ，一般从键盘输出，但也可以代表其他输入，如语音输入。
    <sub>,<sup> 分别代表下标 和上标 。
    <i> 代表一段不同性质 的文字，如技术术语、外文短语等。
    <b> 代表一段需要被关注 的文字。
    <u> 代表一段需要下划线呈现的文本注释，如标记出拼写错误的文字等。
    <mark> 代表一段需要被高亮的引用 文字。
    <ruby> 代表被ruby 注释 标记的文本，如中文汉字和它的拼音。
    <rt> 代表ruby 注释 ，如中文拼音。
    <rp> 代表 ruby 注释两边的额外插入文本 ，用于在不支持 ruby 注释显示的浏览器中提供友好的注释显示。
    <bdi> 代表需要脱离父元素文本方向的一段文本。它允许嵌入一段不同或未知文本方向格式的文本。
    <bdo> 指定子元素的文本方向 ，显式地覆盖默认的文本方向。 dir,这个标签包含文本的文本方向. 属性值可为:
    • ltr: 从左往右写，与现代汉语的书写习惯相同.
    • rtl: 从右往左写，与古代汉语书写习惯相同.
    • auto: 自动，由浏览器决定文字的排列方向.
    <span> 代表一段没有特殊含义的文本，当其他语义元素都不适合文本时候可以使用该元素。
    <br> 代表换行 。
    <wbr> 代表建议换行 (Word Break Opportunity) ，当文本太长需要换行时将会在此处添加换行符。

    HTML5新增的输入类型

    不支持的浏览器会视为输入类型text
    包含数字值的输入字段：type="number" 请输入6~12位数字
    包含颜色的输入字段：type="color"
    包含一定范围内的值的输入字段：type="range"
    包含日期的输入字段：type="date" max="1979-12-31"
    允许用户选择月份和年份：type="month"
    允许用户选择周和年type="week"
    允许用户选择时间（无时区）type="time"
    允许用户选择日期和时间（有时区）type="datetime"
    允许用户选择日期和时间（无时区）type="datetime-local"
    包含电子邮件地址的输入字段type="email"
    用于搜索字段（搜索字段的表现类似常规文本字段）type="search"
    包含电话号码的输入字段type="tel"
    包含URL地址的输入字段type="url"
    提交表单数据至表单处理程序的按钮:type="submit"
    重置表单按钮:type="reset"
    多行输入字段（文本域）：cols="设置宽度字数" rows="设置高度行数"
