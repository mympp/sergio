数组：

			
name.length;//字符串长度
name[0];//索引

name.toUpperCase();//全部变为大写
name.toLowerCase();//全部变为小写

name.indexOf('aname');//返回数字，搜索字符串位置 
name.substring(0,5);//返回指定索引区间的子串
name.slice(0,3);//截取name的部分元素，返回一个新的name，不输入则为复制。

push();//向数组末尾添加元素
pop();//向数组的最后一个元素删除掉

unshift();//往数组的头部添加若干元素
shift();//把数组第一个元素删掉
splice(2,4,'1','2','3','4');//修改数组。从指定索引开始删除若干元素，再从该位置添加若干元素。哪些元素没写则不起作用。

sort();//对数组进行排序。会直接修改数组的元素位置
reverse();/反转数组元素。

concat();//连接多个数组，返回一个新的数组
join('-');//把数组的每个元素用指定字符连接起来。

多维数组
如果数组的某个元素又是一个Array，则可以形成多维数组：
var arr = [[1, 2, 3], [400, 500, 600], '-'];
var x = arr[1][1]; //返回 500

通过JavaScript来操控Web浏览器中的HTML内容和文档的CSS样式
也可以通过事件处理程序（event handler）来定义文档的行为。事件处理程序是一个在浏览器注册的javascript函数，当特定类型的时间发生时浏览器便调用这个函数。通常我们比较关心的事件类型是鼠标点击时间和键盘按键时间（智能手机的触碰事件）。或者完成文档加载后的事件，用户改变窗口大小时间，输入表单数据事件。定义事件处理程序最简单的方法是给HTML的以“on”为前缀的属性绑定一个回调。
